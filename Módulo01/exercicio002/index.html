Parágrafos e Quebras de Linha

Você pode escrever parágrafos em HTML colocando qualquer coisa dentro das tags <p> </p>

Se precisar quebras o texo em algum lugar específico como esse:
basta usar a tag <br> (Não tem fechamento)

Símbolos Especiais

Vamos adicionar alguns símbolos especiais: © ® ™ € £ ¥ ¢ δ Δ ↑

Vamos adicionar alguns emojis: Busca em Emojipedia que utilizam os códigos Unicode:
✨ → (&#x + Código Unicode → Ex.: &#x + 2728)
(nosso < meta charset="UTF-8" > serve também serve p/ emojis → Qualquer símbolo especial)
