Semântica

O HTML5 foca agora em SEMÂNTICA, ou seja, a utilização de elementos que denotam seu significado e função no conteúdo.

O CSS3 foca em FORMATAÇÃO, ou seja, a utilização de elementos que denotam seu significado e função na apresentação do conteúdo. Tags como <center> e <marquee> são obsoletas, pois não possuem significado semântico e não são recomendadas para formatação de conteúdo. O ideal é utilizar sempre as CSS para estilizar o conteúdo, mantendo a semântica no HTML.

Principais Formatações

Destaque e Negrito

O elemento <strong> é utilizado para indicar que o conteúdo é de extrema importância, enquanto o elemento <b> é utilizado para indicar que o conteúdo deve ser destacado visualmente, mas não possui significado semântico. O ideal é utilizar o elemento <strong> para indicar importância e o elemento <b> para destacar visualmente.

Este texto está em strong → Dá sentido SEMÂNTICO para o termo.
Este texto está em bold → Dá destaque VISUAL para o termo.

Ênfase e Itálico

O elemento <em> é utilizado para indicar que o conteúdo é de extrema importância, enquanto o elemento <i> é utilizado para indicar que o conteúdo deve ser destacado visualmente, mas não possui significado semântico. O ideal é utilizar o elemento <em> para indicar importância e o elemento <i> para destacar visualmente.

Este texto está em emphasis → Dá sentido SEMÂNTICO para o termo.
Este texto está em italic → Dá destaque VISUAL para o termo.

Outras Formatações

Texto marcado

O elemento <mark> é utilizado para indicar que o conteúdo deve ser marcado ou destacado, geralmente para fins de destaque visual em um contexto específico.

Textos com tamanho

Texto Deletado

O elemento <del> é utilizado para indicar que o conteúdo foi excluído ou removido.

Texto inserido e sublinhado

Podemos marcar que um texto foi inserido com a tag <ins> e sublinhado com a tag <u>.

Textos Sobre e Subescritos

Com a tag <sup> podemos criar textos sobreescritos como x2 e com a tag <sub> textos subescritos como H2O.
