Exercícios e Desafios - Curso de HTML5 e CSS3

O módulo inicial possui uma carga horária equivalente à 40 Horas e aborda os fundamentos da Web, conceitos de Front-end, as primeiras tags estruturais do HTML5 e a introdução à estilização com CSS3.
